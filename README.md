**Süper Lig Atlası**, Trendyol Süper Lig futbolcularının istatistiklerini, piyasa değerlerini ve form durumlarını detaylı bir şekilde analiz etmenizi sağlayan **SPA (Tek Sayfa Uygulaması)** mimarisiyle geliştirilmiş interaktif bir futbol platformudur. 

Futbolcuların etki skorlarını radar grafikleriyle inceleyebilir, rüya onbirinizi kurabilir ve oyuncuları birbirleriyle detaylı bir şekilde kıyaslayabilirsiniz.

---

## ✨ Özellikler

- 📊 **Detaylı Oyuncu Analizi:** Oyuncuların form, büyük maç performansı, gol/asist katkısı ve genel etki skorlarını hesaplayan özel algoritma.
- 🕸️ **Radar Grafikleri:** Kütüphanesiz (Vanilla SVG) geliştirilmiş, oyuncu verilerini 5 farklı eksende (Hücum, İstikrar, Oyun Aklı, Büyük Maç, Form) görselleştiren profil kartları.
- 🏟️ **Kadro Kur (Squad Builder):** Bütçenize göre en uygun Süper Lig oyuncularını seçerek kendi "Rüya Onbirinizi" oluşturabileceğiniz interaktif simülasyon.
- ⚖️ **Kıyaslama Modu:** İki farklı oyuncunun istatistiklerini ve piyasa değerlerini yan yana, şık bir arayüzle karşılaştırma imkanı.
- 🔍 **Gelişmiş Filtreleme:** Oyuncuları mevkilerine, takımlarına, yaşlarına ve piyasa değerlerine (Slider ve Seçim) göre anlık filtreleme.
- 🌓 **Modern Arayüz & Dark Mode:** Glassmorphism detayları ve akıcı animasyonlarla desteklenen, localStorage entegreli Karanlık/Aydınlık tema seçeneği.

## 🚀 Teknolojiler

Bu proje, harici bir framework veya kütüphane kullanılmadan, saf (Vanilla) web teknolojileriyle geliştirilmiştir:
- **HTML5** & **CSS3** (CSS Variables, Flexbox/Grid, Animasyonlar)
- **Vanilla JavaScript** (ES6+, DOM Manipülasyonu, Event Delegation, Fetch API)
- **SVG** (Dinamik radar grafikleri için)

## 🛠️ Kurulum ve Çalıştırma

Projeyi bilgisayarınızda yerel olarak çalıştırmak oldukça basittir. Herhangi bir Node.js paketi kurmanıza gerek yoktur.

1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/Eren61cyber/superlig-atlasi.git
   ```
2. Proje dizinine girin:
   ```bash
   cd superlig-atlasi
   ```
3. `index.html` dosyasını favori tarayıcınızda açın veya bir Live Server (VS Code vb.) ile başlatın.

> Not: Uygulama içerisinde bazı oyuncu resimleri ve logolar için dış API'lara (Wikipedia vb.) anlık istek atılmaktadır, bu nedenle uygulamanın tam çalışması için aktif bir internet bağlantısı gereklidir.
